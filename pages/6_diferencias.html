Diferencias entre Versiones

Version 1 Version 2
Roles: existen dos tipos de usuario, sin embargo, las funcionalidades para cada uno no estan definidas. Existen 3 roles: Super Admin, Administrador y Común
Estos roles se encuentran bien definidos y cada uno cumple con la funcionalidad requerida.
Salas: se deben crear varias instancias de la aplicación para tener el control de varias salas. Desde la misma plataforma se puede administrar varias salas y asignarles un Administrador a cada sala.
Zonas: las salas no se encuentran asociadas a una zona. Las salas deben estar asignada a una zona, la cual esta relacionada a la Parroquia de un Municipio del Estado.
Incidencias: son configuradas al momento de instalar la plataforma y agregas a través del gestor de base de datos. El Super Administrador puede añardir tantas incidencias como la plataforma lo requiera durante o despues de la instalación.
