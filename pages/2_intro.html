¿Qué es mochuelo, y cómo funciona?

Mochuelo es un Sistema de Alerta Temprana para dispositivos móviles. El sistema de alerta temprana Mochuelo está constituido por dos módulos. Uno de ellos consiste en una aplicación para dispositivo móviles que le permite al usuario enviar un reporte o alerta desde su celular. La otra parte consiste en una interfaz web, operada desde una sala situacional, que permite visualizar y filtrar las alertas para la generación de reportes.
